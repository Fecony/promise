## Installation

Clone the project

```bash
  git clone git@github.com:fecony/promise.git
```

Go to the project directory

```bash
cd promise
```

Then use [nvm](https://github.com/nvm-sh/nvm) to switch to the correct node version

Run with `wasp start`
